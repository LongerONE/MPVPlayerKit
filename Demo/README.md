# MPVPlayerKit Demo

Open `MPVPlayerKitDemo.xcodeproj`, select an iOS Simulator, and run the
`MPVPlayerKitDemo` scheme. The app uses the package's ready-made
`MPVQuickPlayerViewController` and starts a public HLS sample automatically.

Replace `sampleURL` in `AppDelegate.swift` with a media URL containing multiple
audio or subtitle tracks to exercise the track selectors.
